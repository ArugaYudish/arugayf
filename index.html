Biografi Diri Saya

Ditulis oleh Aruga Pada November 2021

Nama saya Aruga Yudish Firmansyah, lahir di kota bandung pada tanggal 30 0ktober 2002. saya adalah anak pertama dari tiga bersaudara. saya memiliki 2 adik laki-laki yang masih bersekolah. Saya sekarang merupakan seorang mahasiswa sistem informasi semester 3 di Institut Teknologi Telkom Purwokerto. Saya memiliki ketertarikan pada bidang IT dan Bisnis. lalu saya juga memiliki beberapa keterampilan seperti skill analisis, coding, komunikasi, negosiasi, kepemimpian, bekerja sama dengan tim, dan berani mengambil keputusan.

Hoby saya yaitu bermain sepak bola, bermain game, coding, stand-up comedy, dan nongkrong di caffe. Motivasi saya yaitu lebih baik terlambat daripada tidak sama sekali. sekarang saya sedang mempelajari bahasa pemrograman C++, java, dan python. Saya juga sekarang sedang memperlajari cara mendesain user interface, mendesain web, merancang sebuah web, membuat sebuah aplikasi. di kampus saya mengikuti beberapa UKM dan Organisasi salah satu Organisasinya yaitu Himpunan Mahasiswa Sistem Informasi (HMSI) dan saya menjadi staff pada divisi Human Resource Development (HRD).

Cerita Paling Berkesan

Ditulis oleh Aruga Pada November 2021

Cerita paling berkesan saya pada saat saya menginjang di bangku SMA. Saya bertemu dengan orang baru, saya bertemu beberapa teman yang sangat fair dan masih berteman sampai saat ini. Hal tersebut bagi saya sangat berkesan karena saya jadi mengerti berpenampilan dan mendapatkan wawasan pergaulan dari sudut pandang yang berbeda. Hal tersebut merupakan sebuah pencapaian terbesasr saya dalam Stand-up comedy.
